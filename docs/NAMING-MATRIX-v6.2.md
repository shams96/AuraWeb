# Isola Vitale — Standardized Product Naming Matrix v6.2
> June 2026 · CONFIDENTIAL · Supersedes all prior nomenclature documents

## Naming Convention Rules

| Rule | Standard |
|------|----------|
| No ingredient names in titles | Function and outcome only — no Hyaluronic Acid, Peptide, Ceramide, Niacinamide in product names |
| Cellular Stage framing | Foundation · Correction · Regeneration · Longevity — no age ranges in consumer copy |
| Optimize, don't fix | 'Renewal' not 'Repair'. 'Defense' not 'Protection'. 'Radiance' not 'Brightening' |
| Tier designation in descriptor only | Clinical (A) and Consumer (B) share same base name; tier shown in SKU code, never the product title |
| ™ on hero innovations | Unicorn Innovations and proprietary systems carry ™. Standard SKUs do not |
| TTW™ in copy, not in name | All product copy references TTW™ performance. Names use outcome-language only |
| Website tone | Names must feel like Skin Intelligence™ protocol steps, not product SKUs |
